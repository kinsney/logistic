
以下的ip就是设备的ip地址，标签上有，默认是：192.168.1.88
第一路继电器指令
http://ip/ecmd?pin set k1 on     -- 打开第1路继电器
http://ip/ecmd?pin set k1 off      关闭第1路继电器
继电器1状态
http://ip/ecmd?pin get k1       返回值 on/off

第二路继电器指令
http://ip/ecmd?pin set k2 on     -- 打开第路继电器
http://ip/ecmd?pin set k2 off      关闭第路继电器
继电器2状态
http://ip/ecmd?pin get k2       返回值 on/off

共16路，以下依次类推。改变kx,x的值即可，x值最大16.

以下是如何用命令修改ip 网关。
拿到设备后，看下设备默认ip地址，一般是192.168.1.88。
如果你路由器不是192.168.1.x的网段，请通过电脑和设备通过网线直接连接，然后配置电脑的ip地址为192.168.1.2，
保存后，打开浏览器，输入设备的默认ip（192.168.1.88），可以看到设备的主页，然后浏览器输入命令
http://默认ip/ecmd?ip 新的ip,比如你需要改的新的ip为192.168.0.99，就输入http://192.168.1.88/ecmd?ip 192.168.0.99
会有返回“ok",不要拔电源，然后修改网关，请确认自己路由器的网关比如你的网关为192.168.0.1，那么就输入http://192.168.1.1/ecmd?gw 192.168.0.1
然后输入重启命令 http://192.168.1.88/ecmd?reset,可以测试是是否修改成功，把电脑的ip改为192.168.0.2，然后浏览器输入
http://192.168.0.99，如果看到主页，说明修改成功。
