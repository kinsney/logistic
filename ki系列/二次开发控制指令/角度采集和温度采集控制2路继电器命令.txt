网络采集adc和温度控制继电器命令如下
协议基于tcp http，默认ip地址192.168.1.77，端口80
以下命令中的空格可以用%20来替代
比如：http://192.168.1.77/ecmd?flag 0   也可以是http://192.168.1.77/ecmd?flag%200

1：选择温度或者角度来控制继电器的闭合和断开

   http://192.168.1.77/ecmd?flag 0               --温度和角度传感器控制继电器无效
   http://192.168.1.77/ecmd?flag 1               --激活温度传感器控制继电器
   http://192.168.1.77/ecmd?flag 2               --激活角度传感器控制继电器  
   http://192.168.1.77/ecmd?rflag                --查看激活状态

2：设置第一路温度门限
   http://192.168.1.77/ecmd?1w get 285ad38a060000fc  --查看第一路的温度值，返回值为实际温度值。
   http://192.168.1.77/ecmd?ch1down xx               --设置第一路温度下限值，xx为十进制值。
   http://192.168.1.77/ecmd?rch1down                 --查看第一路下限值。

   http://192.168.1.77/ecmd?ch1up xx                --设置第一路温度上限值，xx为十进制值。
   http://192.168.1.77/ecmd?rch1up                  -查看第一路上限值。
  
   如果实际测的温度值在上下门限内，继电器闭合，反之断开。

3：设置第二路温度门限
   http://192.168.1.77/ecmd?1w get 28f090af06000039  --查看第二路的温度值，返回值为实际温度值。
   http://192.168.1.77/ecmd?ch2down xx               --设置第二路温度下限值，xx为十进制值。
   http://192.168.1.77/ecmd?rch2down                 --查看第二路下限值。

   http://192.168.1.77/ecmd?ch2up xx                --设置第二路温度上限值，xx为十进制值。
   http://192.168.1.77/ecmd?rch2up                  --查看第二路上限值。
  
   如果实际测的温度值在上下门限内，继电器闭合，反之断开。
  
4：X轴（vo1)角度控制继电器
   http://192.168.1.77/ecmd?anglex                   -- 返回值为x轴角度的电压值（0-5V)对应 （-90°到90°），水平返回值为2.5
   http://192.168.1.77/ecmd?xangledown xxx           -- 设置x轴下限的值，xxx为十进制，比如200，就是2V.
   http://192.168.1.77/ecmd?rxangledown              --查看x轴的下限值
   http://192.168.1.77/ecmd?xangleup xxx             -- 设置x轴上限的值，xxx为十进制，比如400，就是4V.
   http://192.168.1.77/ecmd?rxangleup                --查看x轴的上限值
   如果实际测的温度值在上下门限内，继电器闭合，反之断开。
5: Y轴（vo2)角度控制继电器
   http://192.168.1.77/ecmd?angley                   -- 返回值为y轴角度的电压值（0-5V)对应 （-90°到90°），水平返回值为2.5
   http://192.168.1.77/ecmd?yangledown xxx           -- 设置y轴下限的值，xxx为十进制，比如200，就是2V.
   http://192.168.1.77/ecmd?ryangledown              --查看y轴的下限值
   http://192.168.1.77/ecmd?yangleup xxx             -- 设置y轴上限的值，xxx为十进制，比如400，就是4V.
   http://192.168.1.77/ecmd?ryangleup                --查看y轴的上限值
   如果实际测的温度值在上下门限内，继电器闭合，反之断开。

6：当角度和温度无效时，可以用命令来控制继电器的闭合和断开
    http://192.168.1.77/ecmd?pin set k1 on              -- 继电器第一路闭合
    http://192.168.1.77/ecmd?pin set k1 off             -- 继电器第一路断开
    http://192.168.1.77/ecmd?pin set k2 on              -- 继电器第二路闭合
    http://192.168.1.77/ecmd?pin set k2 off             -- 继电器第二路断开
