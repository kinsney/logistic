以下命令适用H8-8设备

window xp 或者window7,先下载snmputil.exe  工具

MIB库头       .iso.org.dod.internet.private.enterprises 
对应oid数字   .1.3.6.1.4.1


控制第1路继电器-闭合

snmputil get [fe80:0:0:0:2e0:b6ff:fe0e:8888] public .1.3.6.1.4.1.2021.13.23.6.1.1


-断开

snmputil get [fe80:0:0:0:2e0:b6ff:fe0e:8888] public .1.3.6.1.4.1.2021.13.23.6.1.2

 

控制第2路继电器-闭合

snmputil get [fe80:0:0:0:2e0:b6ff:fe0e:8888] public .1.3.6.1.4.1.2021.13.23.6.2.1             


-断开

snmputil get [fe80:0:0:0:2e0:b6ff:fe0e:8888] public .1.3.6.1.4.1.2021.13.23.6.2.2

 

控制第3路继电器-闭合

snmputil get [fe80:0:0:0:2e0:b6ff:fe0e:8888] public .1.3.6.1.4.1.2021.13.23.6.3.1             


-断开

snmputil get [fe80:0:0:0:2e0:b6ff:fe0e:8888] public .1.3.6.1.4.1.2021.13.23.6.3.2

。。。。

 

控制全部8路继电器-闭合

snmputil get [fe80:0:0:0:2e0:b6ff:fe0e:8888] public .1.3.6.1.4.1.2021.13.23.6.9.1             


-断开

snmputil get [fe80:0:0:0:2e0:b6ff:fe0e:8888] public .1.3.6.1.4.1.2021.13.23.6.9.2



获取DHT传感器温度值-返回值 就是温度的值（十进制）

snmputil walk [fe80:0:0:0:2e0:b6ff:fe0e:8888] public .1.3.6.1.4.1.2021.13.23.5.2

 

获取DHT传感器湿度值-返回值就是湿度的值（十进制）

snmputil walk [fe80:0:0:0:2e0:b6ff:fe0e:8888] public .1.3.6.1.4.1.2021.13.23.5.3





以下是linux（ubuntu系统下测试）

   首先在终端 安装snmp程序，命令如下 

    $ Sudo apt-get install snmp                   

    $ snmpget -c public -v 1 [fe80:0:0:0:2e0:b6ff:fe0e:8888] 1.3.6.1.2.1.1.4   --获取系统信息
   
    控制第1路继电器-闭合

$ snmpget -c public -v 1 [fe80:0:0:0:2e0:b6ff:fe0e:8888] 1.3.6.1.4.1.2021.13.23.6.1.1

 -断开

$ snmpget -c public -v 1 [fe80:0:0:0:2e0:b6ff:fe0e:8888] 1.3.6.1.4.1.2021.13.23.6.1.2

。。。。依次类推

 

全部闭合

$ snmpget -c public -v 1 [fe80:0:0:0:2e0:b6ff:fe0e:8888] 1.3.6.1.4.1.2021.13.23.6.9.1

全部断开

$ snmpget -c public -v 1 [fe80:0:0:0:2e0:b6ff:fe0e:8888] 1.3.6.1.4.1.2021.13.23.6.9.2

 

温度获取

$ snmpget -c public -v 1 [fe80:0:0:0:2e0:b6ff:fe0e:8888] 1.3.6.1.4.1.2021.13.23.5.2

$ snmpget -c public -v 1 [fe80:0:0:0:2e0:b6ff:fe0e:8888] 1.3.6.1.4.1.2021.13.23.5.3
