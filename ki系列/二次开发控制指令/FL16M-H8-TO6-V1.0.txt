

以下的ip就是设备的ip地址，标签上有，默认是：192.168.1.99,以标签为准。
继电器共16路，9-16路可以通过外部扩展卡控制。

1，如何读取16路继电器的状态
    http://192.168.1.99/ecmd?i2c rwd 87 11      返回值为word字，例如返回0xffff，为所有继电器闭合，从低位到高位对应1-16路的状态。
2，如何获取8路输入的状态
    http://192.168.1.99/ecmd?i2c rbd 87 10      返回值为byte字节，例如返回0xff，为扩展板输入状态为1，从低位到高位对应1-8输入。
3, 如何激活8路输入扩展
    http://192.168.1.99/ecmd?i2c wbd 87 13 1    激活8路外部扩展板输入关联继电器功能
    http://192.168.1.99/ecmd?i2c wbd 87 13 0    8路外部扩展板输入功能关联继电器无效
    需要重启生效。
4, 如果有云接入(仅限有云账号的用户）
    http://192.168.1.99/ecmd?i2c wbd 80 3 1     激活云设备
    http://192.168.1.99/ecmd?i2c wbd 80 3 0     云账号设备无效
5，继电器控制指令：
   http://ip/ecmd?channel xxx yyy zzz mmm   -- xxx 为0-255的十进制，表示选择前8路，yyy为9-16路，十进制0-255，zzz为十进制，对应前8路的位，mmm为十进制，对应9-16路。
   指令示例
   比如 http://192.168.1.99/ecmd?channel 255 255 255 255   此命令说明16路的继电器全部闭合
        http://192.168.1.99/ecmd?channel 255 255 0 0       此命令说明16路的继电器全部断开

   多用户控制举例解析
                                       __________________8的二进制位00001000,说明第4路继电器被选中。
                                      |   _______________10的二进制为00001010,说明第10路和第12路的继电器被选中.
                                      |  |  _____________8的二进制位00001000,说明第4路继电器闭合，如果此位置为0，即00000000，则第4路断开，其它路状态不变。 
                                      |  | |   __________10的二进制为00001010,说明第10路和第12路的继电器闭合，如果此位置为8，即00001000，则10路断开，12路闭合，其它路状态不变
                                      |  | |  |
     http://192.168.1.99/ecmd?channel 8 10 8 10 
     

     此命令也适合6路的设备，比如控制6路的第一路
                                       __________________4的二进制位00000100,说明第1路继电器被选中，高位的前6位有效，低位的后2位无效。
                                      |   _______________这里的值6路中无效，保持为0即可。
                                      |  |  _____________4的二进制位00000100,说明第1路继电器闭合，如果此位置为0，即00000000，则第1路断开，其它路状态不变。 
                                      |  | |   __________这里的值6路中无效，保持为0即可。
                                      |  | |  |
     http://192.168.1.99/ecmd?channel 4  0 4  0 

            
  6,如果配有温湿度一体传感器，可通过指令获取温湿度值
     http://192.168.1.99/ecmd?dht%20temp   返回值为温度值
     http://192.168.1.99/ecmd?dht%20humid  返回值为湿度值
     
  7,h8-8设备特别备注才有此功能，默认没有此命令，返回值无延时。
    http://ip/ecmd?output                -----返回值为输出状态值
    http://ip/ecmd?input                 -----返回值为输入的值

  8,2路和6路设备如果为点动，可通过命令修改延时时间-2015-7月以后版本
    http://ip/ecmd?dtime xxx         -- xxx为10进制的数，比如你要改为60秒延时，xxx就是60.  
    http://ip/ecmd?dtime 1800        -- 如果设置值大于等于1800，则延时无效。
